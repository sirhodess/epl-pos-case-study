# EPL POS Case Study

A full-stack check-in, inventory, rental, and point-of-sale system built for an electronics prototyping lab as part of a computer science capstone project.

## Overview

The system supports public inventory browsing, staff inventory management, customer records, rentals, returns, refunds, and point-of-sale workflows.

## My Role

I contributed primarily to the frontend and full-stack workflow integration, including public inventory UI polish, customer transaction views, rental/return flows, POS interface improvements, theme support, and API integration.

## Tech Stack

- React
- TypeScript
- FastAPI
- PostgreSQL
- Docker
- GitHub workflows

## Key Contributions

- Built and refined POS rental, return, refund, and customer transaction workflows
- Improved public-facing inventory browsing and responsive layout
- Connected frontend views to backend API data
- Added UI polish, loading states, error handling, and toast feedback
- Helped organize staff-facing workflows for inventory and customer management
